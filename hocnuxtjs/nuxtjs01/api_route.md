# Xây dựng API với Nuxtjs

- URL
- HTTP Method

* GET --> Lấy dữ liệu
* POST --> Thêm mới dữ liệu
* PUT/PATCH --> Cập nhật dữ liệu
* DELETE --> Xóa dữ liệu

- Request:

* Body --> Chỉ áp dụng với post, put, patch, delete
* Headers

- Response

* Body: Nội dung phản hồi của api
* Status Code: Trạng thái
* Headers
